# 📱 ShopPhone - Ứng dụng thương mại điện tử

**ShopPhone** là một ứng dụng thương mại điện tử được phát triển trong năm 2025. Ứng dụng cung cấp trải nghiệm mua sắm điện thoại hiện đại, tiện lợi và thân thiện với người dùng, hỗ trợ cả desktop lẫn mobile.

---

## 🚀 Tính năng chính

- 🔍 Duyệt và tìm kiếm sản phẩm theo danh mục.
- 🖼️ Xem chi tiết sản phẩm với hình ảnh, giá cả và đánh giá.
- 🛒 Quản lý giỏ hàng và đặt hàng trực tuyến.
- 👤 Đăng ký, đăng nhập và quản lý tài khoản người dùng.
- 💻 Giao diện thân thiện, tối ưu cho nhiều thiết bị.

---

## 🛠️ Công nghệ sử dụng

| Công nghệ         | Vai trò                                         |
|-------------------|-------------------------------------------------|
| **ReactJS**       | Xây dựng giao diện người dùng động và hiệu quả  |
| **Redux Toolkit** | Quản lý trạng thái ứng dụng (giỏ hàng, user...) |
| **React Router**  | Điều hướng và quản lý các trang trong ứng dụng  |
| **Axios**         | Gửi HTTP request, giao tiếp với API backend     |
| **Bootstrap**     | Thiết kế responsive và bố cục giao diện         |
| **CSS**           | Tuỳ chỉnh giao diện, hiệu ứng và phong cách     |

---

## 🎯 Mục tiêu dự án

- Xây dựng một nền tảng thương mại điện tử hiện đại, dễ sử dụng.
- Giúp người dùng dễ dàng tìm kiếm và mua sắm các sản phẩm điện thoại.
- Hệ thống có khả năng mở rộng trong tương lai với các tính năng như:
  - Thanh toán trực tuyến.
  - Theo dõi đơn hàng.
  - Tích hợp chương trình khuyến mãi.
